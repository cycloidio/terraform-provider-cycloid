# cycloid_plugin_manager (Resource)

Registers a Plugin Manager with an organization. The Plugin Manager is the runtime service that
executes installed plugins inside the organization. Creating this resource registers the manager
with Cycloid and the manager service in one step.

Each organization has at most one active Plugin Manager. Destroying this resource unregisters the
manager and stops all running plugins in the organization.

Set `wait_until_connected = true` to block the apply until the manager reports `connected` status
(up to 5 minutes). Useful when subsequent resources depend on the manager being ready.


## Example Usage

```terraform
// A plugin manager is the runtime that executes installed plugins. The
// resource registers the manager with the organization and immediately accepts
// the invite (invite_status is always "accepted" once the resource exists).
resource "cycloid_plugin_manager" "default" {
  organization = "my-org"
  name         = "default-plugin-manager"
  url          = "https://plugin-manager.example.com"
}
```
